# AGENTS.md

## 目的

このプロジェクトにおける AI エージェント共通の作業方針を定義する。

## 基本方針

- **会話言語**: 日本語
- **コード内コメント**: 日本語
- **エラーメッセージ**: 英語
- **コミットメッセージ**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従い、説明を日本語で記載する。
- **ブランチ命名**: [Conventional Branch](https://conventional-branch.github.io) に従う。

## 判断記録のルール

- 判断内容の要約、検討した代替案、採用理由、前提条件、不確実性を明示する。

## 開発手順

1. **プロジェクト理解**: 既存のコードベースと技術スタックを分析する。
2. **環境構築**: `yarn install` を行い、依存関係を最新にする。
3. **実装**: プロジェクトの命名規則やアーキテクチャに従い変更を行う。
4. **検証**: `yarn lint` および型チェック（`yarn compile` 等）を実行し、動作を確認する。

## セキュリティ / 機密情報

- 認証情報や API キーを Git にコミットしない。
- ログに秘密情報が出力されないようにする。

## リポジトリ固有

- このプロジェクトは `downloader` と `viewer` の 2 つの主要コンポーネントで構成されている。
- データ共有は `/data` ディレクトリ以下の JSON ファイルで行われる。
- 外部ツール（yt-dlp, ffmpeg, mp3gain / rgain3）に依存している。
